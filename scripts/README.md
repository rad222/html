# RaViewer data parser

### Installation
```sh
$ https://github.com/rad222/html.git
$ cd ~/html/scripts
$ pip install -r requirements.txt
```
### Update from repository
```sh
$ cd ~/html/scripts
$ git pull
```
### Running
```sh
$ cd ~/html/scripts
# run parsing for Spain:
$ python test_eurdep_single.py ES
```
